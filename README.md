# 🏯 八王子城バーチャル案内 (WebGL)

# Hachioji Castle 3D Exploration

本プロジェクトは、八王子城跡を 3D 空間上で自由に探索できるアプリケーションです。
プレイヤー（ロボット）を操作し、史跡内の各地点へ移動しながら地形や遺構の理解を深めることを目的としています。

本バージョンでは、マルチプレイヤー機能に対応し、複数人で同時に探索・会話が可能になりました。

WEBGL ビルドに対応しており、ブラウザ上で操作可能です。

🔗 公開サイト
https://matsumura-shoichi.github.io/Hachioji-jo/

# 🎮 プロジェクト概要

* 実測 DEM データを用いた実寸スケール地形再現
* 三人称視点によるフィールド探索
* 地点ごとの歴史解説表示
* Cinemachine を活用した演出
* WEBGL 対応（ブラウザ操作）
* マルチプレイヤー対応（音声チャット）

---

# 🌐 マルチプレイヤー機能

本プロジェクトは、Photon のネットワークサービスを利用しています。

* Photon PUN（Photon Unity Networking）によるマルチプレイヤー通信
* Photon Voice による音声チャット機能

これにより、複数のプレイヤーが同一空間に参加し、
リアルタイムで会話しながら八王子城跡を探索できます。

---

# 🚀 ゲームの開始方法

1. プレイヤー名を入力
2. 「ログイン」ボタンをクリック
3. 自動的にルームへ接続しゲーム開始

※入力した名前はロボットの頭上に表示され、他プレイヤーにも共有されます。

---

# ⚠️ Web版の注意

※Web版では日本語入力が正常に動作しない場合があります。
プレイヤー名は英数字で入力してください。

※同一ブラウザの複数タブでの同時プレイは正常に動作しない場合があります。
別のブラウザまたは別端末でご利用ください。
